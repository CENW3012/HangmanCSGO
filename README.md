## How to Play

1. The game will select a random word
2. Click letters on the virtual keyboard to guess
3. You have 6 wrong guesses before the game ends
4. Guess all letters correctly to win!

## Setup

1. Clone this repository
2. Open `index.html` in your web browser
3. Make sure all asset files are in the correct directories
